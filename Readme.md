# Doodcopy

Doodcopy is a simple program to copy DoodStream public folder to your folder

## Installation

Just download to your computer.


## Usage

```bash
# Install all requirements from requirements.txt
pip install -r requirements.txt

# Run the program

python3 main.py

```
